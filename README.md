# Revista da ANPEGE — Templates oficiais

Este repositório público disponibiliza os templates oficiais da Revista da ANPEGE para submissão:

- Template de manuscrito (`templates/`)
- Modelo de carta de apresentação (`templates/`)

## Como usar
1. Acesse a pasta `templates/` e baixe o arquivo desejado.
2. Utilize sempre a versão mais recente disponível no repositório.
3. Para dúvidas sobre normas e procedimentos de submissão, consulte o site da revista (OJS).

## Observação
Em caso de divergência entre os arquivos deste repositório e as informações publicadas no site da revista, prevalece o que constar no site.

Site da revista (OJS): https://ojs.ufgd.edu.br/anpege/
